# Classic Load Balancer
- v1 AWS Load Balancer, AWS tags as deprecated
- Supports TCP (Layer 4)
- Supports HTTP and HTTPS (Layer 7)
- Fixed hostname: `<clb-name>-<internal_id>.region.elb.amazonaws.com`
